About This Archive

The Dossier Project documents homicide investigations using publicly available information — court records, law enforcement statements, and verified news reporting. Every case is labeled with its current status: solved, closed, or cold. A separate Unexplained Files section covers ghost reports, cryptid sightings, and urban legends — these are filed as reported accounts, not verified events.

Suspects are named only where law enforcement or courts have publicly identified them. Unproven allegations are marked alleged. If you believe a case file contains an error, contact us for correction.

This is a demo build — all cases currently shown are fictional placeholders for interface design purposes.
