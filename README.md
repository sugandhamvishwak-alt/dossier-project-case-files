# Dossier Project - Case Files

A research database of declassified criminal cases and unsolved mysteries from US and UK law enforcement.

## Purpose
This project compiles publicly available information about historical unsolved and solved criminal cases, providing researchers and true crime enthusiasts with organized, verified information sourced from official law enforcement databases and FBI records.

## Cases Included
- Zodiac Killer (US, unsolved)
- Black Dahlia (US, unsolved)
- Jack the Ripper (UK, unsolved)
- Thames Torso Murders (UK, unsolved)

## Data Structure
All case information is stored in `data/cases.json` in a standardized format including:
- FIR/Police complaint
- Murder story and methodology
- Crime scenes
- Suspect information (redacted family names)
- Evidence and weapons
- Court trials
- Case conclusion
- Official source links

## Sources
All information sourced from:
- FBI Vault (vault.fbi.gov)
- DOJ Records
- Metropolitan Police UK Archives
- Official court documents

## Usage
Case data can be searched and filtered by country, case type, status, year, and other parameters.
